Предварительная версия ядра распознавания для интеграции с FoxPro
=================================================================

1. все необходимые бибиотеки находятся в libs
2. необходимо зарегистрировать библиотеку с COM объектами с помощью register_libs.bat
3. прилагаемый пример для FoxPro запускается выполнением program1.prg
4. для COM открыты пока только базовые возможности (доступны только методы start, stop, setup, init, deinit класса RecarKernel), функционал будет допоняться.

См. файл помощи help\recar2.core.chm.
